

# PayPayAutoPriceDown（PayPayPriceDown）
PayPayフリマアプリの商品編集画面において、
価格の値下げ操作を自動化する Android アクセシビリティサービス です。

## 概要
本アプリは、PayPayフリマの 商品詳細ページを起点 として、以下の操作を自動で実行します。
- 「編集する」ボタンの自動タップ
- 編集画面への遷移
- 画面スクロール
- 商品価格を 200円値下げ
- 「保存する」ボタンのタップ
- 商品詳細ページへ戻る
※ 画面遷移自体（商品一覧 → 商品詳細）は手動操作となります。

---

## 対応環境
- Android 13 以上（推奨：Android 14 / 15 / 16）
- Pixel シリーズで動作確認済み
- PayPayフリマ公式アプリ（最新版）

---

## インストール手順
1. APK ファイルを端末にダウンロード
2. 初回インストール時、
   「このアプリのインストールを許可」を有効化
3. インストール完了後、アプリを起動

---

## 初期設定（必須）
### ① アクセシビリティを有効化
以下の手順で有効にしてください。
```text
設定 → ユーザー補助 → ダウンロードしたアプリ
→ PayPayAutoPriceDown → 使用を許可
```

※ 初回は「制限付きの設定」が表示される場合がありますが、
　表示される案内に従ってすべて許可してください。

### ② アクセシビリティのショートカット設定（推奨）
ショートカットを設定することで、
画面遷移なしでスムーズに値下げ操作が可能 になります。
```text
設定 → ユーザー補助 → PayPayAutoPriceDown
→ ショートカットを有効化
```
※ ナビゲーションバーまたはフローティングボタンに表示されます。

## 使い方
### 基本的な操作手順
1. PayPayフリマアプリを起動
2. 「出品した商品一覧」を表示
3. 値下げしたい商品をタップし 商品詳細ページを表示
   ⚠️ この時点では「編集する」ボタンは押さないでください
4. PayPayAutoPriceDown の
   「値下げ開始」またはショートカットボタンをタップ
    画面上に「準備中」と表示されます（正常です）
5.そのまま商品詳細ページを表示した状態で待機

→ 自動で以下が実行されます
- 編集するボタンタップ
- 編集画面へ遷移
- 価格を200円値下げ
- 保存
- 商品詳細ページへ戻る

## 重要な仕様・注意点
### ■ 編集画面からは開始されません

本ツールは
「商品詳細ページ（編集するボタンが表示されている画面）」を開始地点
として設計されています。
すでに編集画面が表示されている状態では、
自動処理は開始されません。

### ■ 連続実行について（重要）
連続して値下げを行う場合は、
商品一覧ページへ戻ったタイミングで以下の操作が必要です。
```text
ショートカットボタンを
ON → OFF → ON
（2回押す）
```
これにより次の商品で再度自動処理が実行されます。

### ■ 動作が始まらない場合
以下をご確認ください。
- 商品詳細ページで待機しているか
- 編集画面から開始していないか
- アクセシビリティが ON になっているか
- 一度アクセシビリティを OFF → ON し直す

## 補足（動作安定性について）
端末によっては画面遷移時のアニメーションにより
検知タイミングがずれる場合があります。

可能であれば、以下設定を推奨します。

```text
開発者オプション →
・ウィンドウアニメーションスケール：オフ
・トランジションアニメーションスケール：オフ
・Animator 再生時間スケール：オフ
```
※ 必須ではありませんが、安定性が向上します。
